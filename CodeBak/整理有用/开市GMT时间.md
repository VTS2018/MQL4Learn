# 外汇三大交易时段（GMT 标准时间）

您的服务器时区是 **GMT+2**，以下是各大交易时段在 **GMT 标准时间**下的开市/闭市时间：

---

## 📊 三大时段时间表（GMT 时间）

### **1. 亚洲盘（东京为主）**
```
冬/夏令时通用：
- 开市：GMT 00:00
- 闭市：GMT 09:00
- 持续时间：9小时
```

**说明：**
- 东京当地时间 9:00-18:00（JST = GMT+9）
- 包含悉尼、东京、新加坡、香港等市场

---

### **2. 欧洲盘（伦敦为主）**
```
冬令时（10月-3月）：
- 开市：GMT 08:00
- 闭市：GMT 17:00

夏令时（3月-10月）：
- 开市：GMT 07:00
- 闭市：GMT 16:00
```

**说明：**
- 伦敦当地时间 8:00-17:00
- 欧洲盘是全球外汇交易量最大的时段
- 包含伦敦、法兰克福、巴黎等市场

---

### **3. 美国盘（纽约为主）**
```
冬令时（11月-3月）：
- 开市：GMT 13:00
- 闭市：GMT 22:00

夏令时（3月-11月）：
- 开市：GMT 12:00
- 闭市：GMT 21:00
```

**说明：**
- 纽约当地时间 8:00-17:00（EST = GMT-5 或 EDT = GMT-4）
- 与欧洲盘有重叠时段（波动最剧烈）

---

## 🔥 黄金交易时段（重叠区）

**欧美盘重叠（波动最大）：**
```
冬令时：GMT 13:00-17:00（4小时）
夏令时：GMT 12:00-16:00（4小时）
```
这是外汇市场流动性最强、波动最大的时段！

---

## 📝 转换到您的服务器时间（GMT+2）

如果要在代码中使用，需要加 2 小时：

| 时段 | GMT 时间 | 您的服务器时间（GMT+2） |
|------|---------|----------------------|
| **亚盘开市** | 00:00 | 02:00 |
| **亚盘闭市** | 09:00 | 11:00 |
| **欧盘开市（冬）** | 08:00 | 10:00 |
| **欧盘开市（夏）** | 07:00 | 09:00 |
| **美盘开市（冬）** | 13:00 | 15:00 |
| **美盘开市（夏）** | 12:00 | 14:00 |

---

## ⚠️ 夏令时注意事项

- **欧洲夏令时**：3月最后一个周日 - 10月最后一个周日
- **美国夏令时**：3月第二个周日 - 11月第一个周日
- 两者不完全同步，会出现 2-3 周的"混合期"

**建议：** 在代码中设置 `Server_Time_Offset = -2`（因为您的服务器是 GMT+2）即可自动适配！
